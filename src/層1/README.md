# 層1：共通処理

## 概要

層1は、層0で定義されたエンティティに対する**共通的な基本操作**を提供するレイヤーです。

すべてのTCGタイトルで共通する処理（状態管理、数値管理、ゾーン操作など）を実装し、上位層（層2以降）でのTCG固有ルール実装の基盤となります。

## 責務

層1が定義するもの：
- ✅ エンティティの基本的な操作（状態追加・削除、数値変更、分類管理）
- ✅ ゾーンへのエンティティの追加・取り出し・移動
- ✅ 汎用的なユーティリティ関数
- ✅ すべてのエンティティに対する統一されたインターフェース

層1が定義しないもの：
- ❌ ゲーム固有のルール
- ❌ ターン進行・フェイズ管理
- ❌ 勝敗判定
- ❌ カード効果の処理
- ❌ 具体的なゲーム処理（デッキ、手札、墓地など）

## モジュール一覧

### 共通処理（Common Utilities）
すべてのモジュールから利用される汎用ユーティリティ関数。

**ファイル:**
- `共通処理/共通処理.ts` - 実装
- `共通処理/共通処理.md` - 仕様書

**提供機能:**
- ランダム生成（整数生成、シャッフル）
- 配列操作（検索、挿入、削除、取り出し）
- 集計（個数取得、判定）
- ログ出力

**例:**
```typescript
import { ランダム整数, シャッフル, 検索 } from '層1';

const randomNum = ランダム整数(1, 100);
const shuffled = シャッフル([1, 2, 3, 4, 5]);
const found = 検索(array, item => item.id === 'xyz');
```

### ゾーン処理（Zone Operations）
ゾーンに対する基本操作を提供します。

**ファイル:**
- `ゾーン処理/ゾーン処理.ts` - 実装
- `ゾーン処理/ゾーン処理.md` - 仕様書

**提供機能:**
- エンティティの追加・取り出し・移動
- ゾーン情報の取得
- ゾーンのクリア・判定

**例:**
```typescript

const updatedZone = エンティティを追加(zone, entity);
const [fromZone, toZone] = エンティティを移動(zone1, zone2, entityId);
```

## 設計原則

### 不変性（Immutability）
すべての操作は元のオブジェクトを変更せず、新しいオブジェクトを返します。

```typescript
// ❌ 非推奨：ミューテーション
card.name = '新しい名前';

// ✅ 推奨：新しいオブジェクトを返す
const updatedCard = 名前を変更(card, '新しい名前');
```

### 統一されたインターフェース
カード・プレイヤー・ゲームオブジェクトは同じ操作パターンを持ちます。

```typescript
// すべて同じパターン
カード処理.状態を追加(card, '状態名');
プレイヤー処理.状態を追加(player, '状態名');
ゲームオブジェクト処理.状態を追加(gameObj, '状態名');
```

### 汎用性
すべての操作はTCG依存性がありません。複数のTCGタイトルで同じ実装を使用できます。

## 使用方法

### 全関数をインポート
```typescript
import {
  ランダム整数,
  シャッフル,
  ゾーン処理,
  カード処理,
  プレイヤー処理,
  ゲームオブジェクト処理
} from 'src/層1';
```

### 名前空間でインポート
```typescript
import * as Layer1 from 'src/層1';

Layer1.ランダム整数(1, 10);
Layer1.カード処理.状態を追加(card, 'state');
```

## 層2・層3への拡張

層1の操作を組み合わせて、TCG固有のルールを実装します：

**層2の例：**
```typescript
// 層1の操作を使用して、具体的なゲームロジックを実装
function 攻撃する(attacker: カード, defender: カード) {
  const power = カード処理.数値を取得(attacker, 0);
  // 攻撃ロジックの実装...
}
```

**層3の例：**
```typescript
// 具体的なカード定義を層1の操作で構築
const 火の玉カード = {
  ...カード処理.名前を変更({...}, '火の玉'),
  // 具体的なカード情報...
};
```
